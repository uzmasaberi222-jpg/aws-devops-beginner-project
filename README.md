# 🚀 AWS DevOps Beginner Project

This project demonstrates my learning journey in AWS and DevOps by deploying a simple web application on a cloud server.

---

## 📌 Project Objective
To understand how to launch, configure, and deploy applications using AWS cloud services.

---

## 🛠️ Tools & Technologies
- AWS EC2
- Linux (Ubuntu)
- Nginx
- GitHub

---

## ⚙️ Steps Performed
1. Created an EC2 instance on AWS  
2. Connected to the server using SSH  
3. Installed Nginx web server  
4. Deployed a sample website  
5. Verified deployment via browser
